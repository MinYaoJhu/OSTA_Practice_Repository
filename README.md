# OSTA_Practice_Repository

## Overview

This repository is a **personal learning and practice environment** based on the book:

**“Orchestrating Spatial Transcriptomics Analysis with Bioconductor (OSTA)”**

It is designed to:
- Reproduce analyses presented in the book
- Test and adapt workflows on local datasets
- Build intuition for spatial transcriptomics analysis using Bioconductor

---

## Notes

- Code is adapted from the official OSTA repository:
  https://bioconductor.org/books/OSTA/
- Some steps may be simplified or modified for learning purposes
